1.4 Representing Real Numbers on the Number Line

Definition

Representing real numbers on the number line helps visualize the order and magnitude of real numbers.

Using the Number Line

The number line extends infinitely in both directions. Each point on the number line corresponds to a unique real number.

  • Positive numbers: Plotted to the right of zero, increasing as we move to the right.
  • Negative numbers: Plotted to the left of zero, decreasing as we move to the left.
  • Zero: Represents neither positive nor negative numbers and acts as the origin.

Properties

  • Real numbers can be represented on the number line in a one-to-one correspondence.
  • The distance between two points on the number line represents the absolute difference between the corresponding real numbers.
  • Operations like addition and subtraction can be visualized as movements along the number line.

Answer 3: The absolute difference is |3 - (-5)| = 8.
